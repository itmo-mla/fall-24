# Лабораторная работа №1 "Алгоритмы кластеризации"

В рамках данной работы был реализован алгоритм иерархической кластеризации, EM-алгоритм и DBSCAN. На выбранных данных была проверена работоспособность этих алгоритмов. Было произведено сравнение разработанных решений и их библиотечных версий. Мои реализации работают за сравнимое с библиотечными время.
